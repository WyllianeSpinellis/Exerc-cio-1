# Exerc-cio-1
1. ÁREA DE UM TERRENO Um pedreiro precisa calcular a área de um terreno retangular para fazer um orçamento. Crie um programa que leia a largura e o comprimento do terreno e mostre sua área. Entradas: largura, comprimento Saída: área do terreno Exemplo de entrada: Largura: 8 Comprimento: 25 Saída esperada: Área do terreno: 200.0 m²
